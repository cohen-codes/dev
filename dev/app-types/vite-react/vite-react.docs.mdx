A React Vite application type for Bit. Supporting SSR and outputs a nodejs server ready output.

## Create an App

Create the app using the Vite app template.

```bash
bit fork bitdev.react/examples/vite-app my-app
bit use my-app
```

You can also add a `my-app.react-vite.ts` file extension to an existing component. See [API reference below](#reference). 

## Run the app

You can list all apps on your workspace using `bit app list`. To run the new app:

```bash
bit run my-app
```

To build the app use:

```bash
bit build my-app
```

## Deploy the app

You can use one of the [Bit deployers](https://bit.dev/docs/getting-started/collaborate/deploy-apps):

- [Netlify](https://bit.cloud/teambit/cloud-providers/deployers/netlify)
- [Vercel](https://bit.cloud/teambit/cloud-providers/deployers/vercel)

If using Ripple CI, don't forget to set your access tokens on the [Ripple CI settings](https://bit.dev/reference/ci/ripple-ci#ripple-ci-secrets).
